Key Design Decision

Dual Session Strategy

FastAPI routes use asyncpg (async SQLAlchemy). MCP tools inside executor threads use psycopg2 (sync). Two pools, two drivers — no blocking, no deadlocks.

Key Design Decision

ContextVar Propagation

The run_id ContextVar is cloned via copy_context() before entering the executor — every log line throughout the full 5-agent run carries the same run_id for correlation.
