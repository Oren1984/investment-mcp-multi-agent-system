# streamlit-ui-skeleton

## Purpose
This skeleton provides a clean starting point for Streamlit-based AI user interfaces.

It is intended for:
- AI demos
- internal tools
- proof of concept projects
- lightweight dashboards
- RAG demo interfaces
- quick model showcase apps

## Structure
- `app/` → main Streamlit application logic
- `app/pages/` → multi-page views
- `app/components/` → reusable UI sections
- `app/services/` → backend/API communication layer
- `app/utils/` → utility helpers
- `assets/` → images, icons, static resources
- `config/` → app configuration guidance
- `docs/` → UI notes, flows, screenshots, planning
- `tests/` → UI or smoke-test related documentation

## Notes
This skeleton is UI-first.
Business logic should remain outside the UI when possible.

Use this template when speed matters more than frontend complexity.
