# RAG Layer

## Purpose
This folder contains the core building blocks of the Retrieval-Augmented Generation system.

It represents the actual internal AI workflow that powers document ingestion, retrieval, ranking, grounding, and response preparation.

## Main Components
- Ingestion
- Chunking
- Embeddings
- Vector storage
- Retrieval
- Reranking
- Grounding
- Prompt construction
- Evaluation

## Notes
This folder is the heart of the project.  
It should contain the real RAG logic, separated clearly by responsibility.
